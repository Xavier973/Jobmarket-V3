# Operations

## Configuration
Les variables d'environnement sont centralisees dans .env (voir config/.env.example).

## Secrets
Ne pas committer les identifiants France Travail. Utiliser un fichier .env local.

## Logging
Les logs d'ingestion sont ecrits dans data/logs/ par source.

## Observabilite
Ajouter par la suite une metrique de volume de collecte et d'erreurs par source.
